# ウィンドウ・バッファ

## 分割・基本操作

| キー | 動作 | 説明 |
|------|------|------|
| `<LocalLeader>vs` | 水平分割 | `:sp` |
| `<LocalLeader>vv` | 垂直分割 | `:vs` |
| `<LocalLeader>T` | 新しいタブへ移動 | `<C-w>T` |
| `<Leader><Space>` | 直前のウィンドウへ | `<C-w>p` |
| `<Leader>o` | アウトライン表示 | `:Outline` |
| `<C-l>` | 再描画 | ハイライト解除＋再描画（web再計算要求） |

## ウィンドウ管理モード（`<Leader>s`）

`<Leader>s`（= `ss`）でモード開始。以降は単キーで連続操作。

### 移動
| キー | 動作 |
|------|------|
| `h` / `j` / `k` / `l` | 左 / 下 / 上 / 右のウィンドウへ |
| `w` | 次のウィンドウへ |

### 配置・サイズ
| キー | 動作 |
|------|------|
| `H` / `J` / `K` / `L` | ウィンドウを左/下/上/右へ移動 |
| `>` / `<` | 横幅を拡大 / 縮小 |
| `+` / `-` | 高さを拡大 / 縮小 |
| `=` | 高さ・幅を均等化 |
| `|` | 最大幅にする |
| `x` | カレントと次を入れ替え |
| `r` / `R` | ウィンドウを下/上向きに回転 |

### 開閉・タブ・バッファ
| キー | 動作 |
|------|------|
| `c` | ウィンドウを閉じる |
| `o` | 他のウィンドウを閉じる |
| `t` | 次のタブへ（`gt`） |
| `T` | ウィンドウを新しいタブへ |
| `b` / `B` | 前 / 次のバッファへ |

## バッファ移動（Bufferモード）

| キー | 動作 |
|------|------|
| `<LocalLeader>b` | 前のバッファへ＋モード開始 |
| `<LocalLeader>B` | 次のバッファへ＋モード開始 |
| （モード中）`b` / `B` | 前 / 次のバッファへ |
| `<Leader>b` | `:Telescope buffers` バッファ一覧 |

## ファイル・ファイラ

| キー | 動作 |
|------|------|
| `<LocalLeader>o` | `:Oil` ファイラ |
| `<Leader>e` | `:Telescope file_browser`（現在のディレクトリ） |
| `<Leader>E`（n） | 現在のバッファを explorer.exe で開く |
| `<LocalLeader>E` | `:e .` カレントディレクトリを開く |

---
💡 **Tips**:
- 連続でウィンドウを動かすなら `<Leader>s` モードに入りっぱなしが速い
- マイナーモード中は `?` でヘルプ、`q`/`Esc` で抜ける
- window-local オプション（wrap/number 等）のトグルはフロートメニューと相性が悪い → グローバル変数管理
