# プラグイン

各カテゴリの専用シートに載らない主要プラグインのキーバインドまとめ。

## Yanky（ヤンク・ペースト）

| キー | 動作 |
|------|------|
| `p` / `P` | ヤンク後 / 前にペースト |
| `gp` / `gP` | ペースト＋カーソルを末尾へ |
| `<C-n>` / `<C-p>` | ヤンク履歴を次 / 前へ循環 |
| `<Leader>p` | `:Telescope yank_history` 履歴一覧 |
| `x` / `X` | レジスタを汚さない文字削除 |

## dial.nvim（数値・文字列の増減）

| キー | 動作 |
|------|------|
| `+` / `-`（n） | カーソル位置をインクリメント / デクリメント |
| `+` / `-`（v） | 選択範囲を増減 |
| `g+` / `g-`（v） | 連番作成（累積増減） |

## flash.nvim（ラベルジャンプ）

| キー | 動作 |
|------|------|
| `<LocalLeader><Space>`（n/x/o） | 入力文字のマッチにラベル→ラベルキーで飛ぶ |

- 表示中の全ウィンドウが対象。`/`(incsearch-migemo)・`f/F/t/T`(clever-f) は奪わない設定。
- migemo（toggle: m）ON でローマ字入力→日本語へジャンプ（cmigemo を `search.mode` に接続）。
- 旧 EasyMotion から移行（Noice 競合のため）。

## ナビゲーション

| キー | 動作 | プラグイン |
|------|------|------------|
| `<C-,>` / `<C-.>` | 前 / 次の位置へ | milfeulle |
| `<C-j>` / `<C-k>` | 下 / 上の空行へ | edgemotion |
| `g;` / `g,` | 前 / 次の変更位置へ（+モード） | 変更リスト |

## surround（nvim-surround）

| キー | 動作 |
|------|------|
| `ys{motion}{char}` | モーション範囲を囲む |
| `yss{char}` | 行全体を囲む |
| `ds{char}` | 囲みを削除 |
| `cs{old}{new}` | 囲みを変更 |
| `S{char}`（v） | 選択範囲を囲む |

## 整列・変換

| キー | 動作 | プラグイン |
|------|------|------------|
| `ga` | テキスト整列 | EasyAlign |
| `<LocalLeader>k` | 大文字小文字変換ループ（+モード） | ConvertCase |
| `<LocalLeader>t` | 翻訳（カーソル下/選択） | translate |
| `ms` / `mS` | 引数/要素を交換 / 選択交換 | ISwap |
| `ciy` / `ciY` | 単語をヤンク内容に置換 | — |

## ブラウザ・外部

| キー | 動作 |
|------|------|
| `ma` / `mA`（n/x） | URLを開く / Web検索（open-browser） |
| `<F8>j` | Junkファイルを開く |
| `,,v` | プレビュー（md/typst/svg/csv/stl/dxf） |

## マーク

| キー | 動作 |
|------|------|
| `mm` | マークをトグル（marks.nvim） |
| `<Leader>m` | `:Telescope marks` マーク一覧 |

## howm / telekasten（`,,` プレフィックス）

| キー | 動作 |
|------|------|
| `,,,` | コマンドパレット（Telekasten panel） |
| `,,c` | 新規メモ作成 |
| `,,n` | 日記エントリ作成（時刻ベース・HowmDiary） |
| `,,d` / `,,w` | 今日 / 今週のノート |
| `,,g` / `,,f` | 全文検索(grep) / ノート一覧 |
| `,,l` / `,,i` | リンクを辿る / リンク挿入 |
| `,,b` / `,,t` | バックリンク / タグ一覧 |
| `,,x` | ToDo 切替 |
| `,,k` | カレンダー表示 |
| `,,a` / `,,A` | come-from 宿主へ飛ぶ / 宣言`<<<`挿入 |
| `,,p` / `,,e` | 画像/draw.io 貼り付け / draw.io 再編集 |
| `,,v` | プレビュー |

## Lazy / Mason / Treesitter（`<F8>` プレフィックス）

| キー | 動作 |
|------|------|
| `<F8>l` / `<F8>s` | Lazy 起動 / 同期 |
| `<F8>u` / `<F8>c` | プラグイン更新 / クリーン |
| `<F8>m` | Mason（LSP管理） |
| `<F8>t` | `:TSUpdate` Treesitter更新 |

---
💡 **Tips**:
- 連番を振るなら選択して `g+`（dial）
- 引数の並べ替えは `ms`（ISwap）、整列は `ga`（EasyAlign）
- メモ・日記まわりは全部 `,,` プレフィックスに集約
